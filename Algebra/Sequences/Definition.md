A sequence is simply a set of ordered objects. A sequence can be infinite or finite in the number of its objects, though generally a sequence contains numbers.
The most general way to show a sequence is to use its objects,
as example: $a_1,a_{2},a_{3},\cdots,a_n$ is a **finite** sequence while $b_1,b_2,b_3,\cdots$ is an **infinite** sequence.(In finite sequences, three dots $\cdots$, means there's some terms between the shown terms, while in infinite sequences those three dots,$\cdots$ , mean it continues infinitely and there's no end.)
The formal way to show a sequence is $\{a_n\}$ where $n$ is a [natural number](sets#Natural-numbers).
Although a sequence can be a ordered set of random numbers, we usually use sequence for numbers which follow a general formula, or in other hand, we use sequence as a function which its input is a natural number. In such sequences, given the formula, we can find the $n$th term without having other terms.
##### Examples:
 1. $\{a_n\}=2,4,6,8,\cdots$
 2. $\{b_n\}=\frac{1}{2},\frac{2}{3},\frac{3}{4},\cdots$
 3. $\{c_n\}=-5,-3,-1,1,3,\cdots$
 4. $\{d_n\}=1,2,4,8,...$
 In example 1, we can see $a_1=2,a_2=4,\cdots$ and more generally $a_n=2n \rightarrow \{a_n\}=\{2n\}$ and $\{2n\}=2,4,6,8,\cdots$. For the rest we have: $b_n=\frac{n}{n+1},c_n=2n-7,d_n=2^{n-1}$. If someone ask us to find $c_{200}$, we can simply use formula $c_{200}=2(200)-7=393$ without having to calculate each term.![[{2n}.jpg]]

##### Special sequences:
  Some sequences which have specific formulas or special usages usually have a name, as example we call $\{an+b\},(a,b∈\mathbb{R})$a [linear sequence](Arithmetic%20sequences(Linear).md) , while $\{an²+bn+c\}$,$(a,b,c∈\mathbb{R})$ is a [quadratic sequence](Quadratic%20sequences.md) and $\{ar^n\}$ is a [geometric sequence](geometric%20sequences(Exponential)).
