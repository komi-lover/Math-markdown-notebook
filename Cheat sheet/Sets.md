#### Natural numbers:
 $$
\mathbb{N}=\{1,2,3,...\}
$$
#### Integers:
 $$\mathbb{Z}=\{a-b|a,b∈\mathbb{N}\}=\{0,\pm 1,\pm 2,\cdots\}$$
#### Rational numbers
 $$\mathbb{Q}=\{\frac{a}{b}|a,b∈\mathbb{Z}\}$$

#### Irrational numbers:
 $$\mathbb{Q}'=\mathbb{R}-\mathbb{Q}=\{\sqrt{2},π,\sin(1),...\}$$

#### Real numbers($\mathbb{R}$):
 This set does not have a unique, simple definition, so here we introduce some of its definitions:

1. A cauchy sequence: a real number is a number that can be represented as the limit of a sequence where a digit is added at the end in each term. for example: $3, 3.0, 3.00, 3.000, ...$ which continues infinitely and obviously is 3, so $3 ∈ \mathbb{R}$ or $1,1.4,1.41,1.412,...$ which approaches $\sqrt{2}$ so $\sqrt{2}∈\mathbb{R}$.
‌
2. Axiomatic: $\mathbb{R}$ is defined to be the unique, complete, ordered field where you can

-  Add, subtract, multiply, and divide number(unless zero for dividing)
-  you can compare numbers, assuming $a,b$ are real numbers, only one of the following is true: $a>b,a=b,a<b$
- there are no holes, which means if $a,b,(a<b)$ are two real numbers, there's always $c∈\mathbb{R}$ where $a<c<b$.

  Using these rules we find out $\mathbb{R}$ is like a complete version of $\mathbb{Q}$ since there are no $p∈\mathbb{Q}$ to satisfy both inequality $p^2≥2, 2≥p^2$ but there is $\sqrt{2}∈\mathbb{R}$ which satisfy both inequalities.
  
3. Any number that can measure a one dimensional unit like length or temperature. For example a square of area $5$ has the length $\sqrt{5}$ ![[Real numbers on length.jpg]]

#### Complex numbers: 
 $$ \mathbb{R²}=\mathbb{C}=\{a+bi|a,b∈\mathbb{R}\}$$
 Here we have $i=(0,1)$ which is usually expressed as $i²=-1$.
 Remember $\mathbb{C}$ is not ordered(meaning if $z_1,z_2$ are two complex numbers, statements like $z_1>z_2$ and $z_1<z_2$ are meaningless)

#### Algebraic number:
 let $P(x)$ be a Polynomial of degree $n∈\mathbb{N}$ with coefficients $a_1,a_2,a_3,...,a_{n}∈\mathbb{Z}$  then $s$ is algebraic if and only if $P(s)=0$.
 Examples: All Rational numbers are algebraic, assume $a,b∈\mathbb{Z}$ then $\frac{a}{b}∈\mathbb{Q}$ is the root of $bx-a=0$. $\sqrt{2}$ is algebraic since it's a root of $x^2-2=0$. $i$ is also algebraic since it's a root of $x^2+1=0$

#### Transcendental numbers:
 Numbers which are not algebraic, as example, by the Lindemann–Weierstrass theorem, $e^a$ is Transcendental whenever $a$ is algebraic.
